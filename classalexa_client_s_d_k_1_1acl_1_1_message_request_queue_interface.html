Detailed Description

An interface that abstracts the operations of a MessageRequestQueuesMap between the standard version and the synchronized implementation.

◆ dequeueRequest()

virtual std::shared_ptr<avsCommon::avs::MessageRequest> alexaClientSDK::acl::MessageRequestQueueInterface::dequeueRequest ( )
pure virtual

Dequeues the next available MessageRequest by taking into account if the queue is waiting for a response. This method keeps track of the queue being processed and once a request from the queue is dequeued moves it to the next queue. It also loops back to the starting queue at the end and exits the loop if a valid MessageRequest is found or if it reaches the place it started.

Returns
MessageRequest if an available, else return nullptr.

Implemented in alexaClientSDK::acl::MessageRequestQueue, and alexaClientSDK::acl::SynchronizedMessageRequestQueue.
