Detailed Description

Class to encapsulate the capability configuration implemented by a capability agent. This is entered as a key/value pair in the map. key: CAPABILITY_INTERFACE_TYPE_KEY, value: The interface type being implemented. key: CAPABILITY_INTERFACE_NAME_KEY, value: The name of the interface being implemented. key: CAPABILITY_INTERFACE_VERSION_KEY, value: The version of the interface being implemented. key: CAPABILITY_INTERFACE_CONFIGURATIONS_KEY, value: A json of the configuration values for the interface being implemented.
