virtual void alexaClientSDK::capabilitiesDelegate::DiscoveryEventSenderInterface::addDiscoveryStatusObserver ( const std::shared_ptr< DiscoveryStatusObserverInterface > &  observer)
pure virtual

This method adds an DiscoveryStatusObserver. The observer gets notified of SUCCESS or FAILURE responses during discovery.

Note
: If there is a failure to send a Discovery event, the onDiscoveryFailed() method will be called immediately. If the discovery event is sent successfully, the onDiscoverySuccess() method will be called only after all the events are successfully sent.
Parameters
observerThe pointer to the observer that will be added.
